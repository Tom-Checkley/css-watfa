Other methodologies

OOCSS

Object Orientated CSS

Like BEM this is concerned with splitting out objects and their elements. And creating slightly more generic objects for example:

          
  #sidebar h3, #footer h3 {
    font-family: Arial, Helvetica, sans-serif;
    font-size: 2em;
    line-height: 1;
    color: #777;
    text-shadow: rgba(0, 0, 0, .3) 3px 3px 6px;
  }

  #footer h3 {
    font-size: 1.5em;
    text-shadow: rgba(0, 0, 0, .3) 2px 2px 4px;
  }
https://www.smashingmagazine.com/2011/12/an-introduction-to-object-oriented-css-oocss/

SMACSS

Scalable Modular Architecture for CSS

SMACSS (pronounced smacks) is about categorising CSS thus creating different classes for:

  1. Base
  2. Layout
  3. Module
  4. State
  5. Theme
http://smacss.com/
