Atomic CSS and CSS Frameworks

Atomic CSS is the idea of using one class name to do one thing e.g:

.p-1 {
    padding: 0.25rem;
  }

This idea is used by many front end frameworks, probably the most well known and popular being Bootstrap or Foundation, though there are plenty more.
