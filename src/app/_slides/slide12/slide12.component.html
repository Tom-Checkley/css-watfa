Tailwind CSS

What is Tailwind?

  • Tailwind is to also a framework that relies on atomic style class names.
  • In it's regular forms it is one of the largest CSS frameworks going file size wise.

So what are it's advantages?

  • It is very simple to customize, thus avoiding the generic look that other frameworks give.
  • When used in a build system configured correctly, unused class names can be removed from the code. Vastly reducing file size

It should be noted that other frameworks can be set up with similar build processes. They are also customizable but no where near as easily.
