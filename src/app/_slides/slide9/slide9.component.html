Pros and Cons of Frameworks

Pros

  • Very quick and easy to use.
  • Out of the box styling.
  • A column based responsive grid system.
  • UI components built in, e.g. dropdown nav items

Cons

  • Very popular, so the site you are building is likely to look quite generic.
  • The amount of class names needed to create all the different perutations leads to large file sizes.
  • For the UI components it will also have a dependancy on jQuery.
